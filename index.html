SpotBugs - Find Bugs in Java Programs

This is the web page for SpotBugs, a program which uses static analysis to look for bugs in Java code.  It is free software, distributed under the terms of the Lesser GNU Public License.

SpotBugs is the spiritual sucessor of FindBugs, carrying on from the point where it left off with the support of its community.

FindBugs requires JRE (or JDK) 1.7.0 or later to run.  However, it can analyze programs compiled for any version of Java, from 1.0 to 1.8.
